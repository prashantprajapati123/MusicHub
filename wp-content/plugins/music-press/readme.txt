Music Press - Manager your store: Genre, Artist, Albums. Supported audio and video file.

== Description ==
Music Press Plugin help you create and manager your music store.
Genre manager, Artist manager, Albums manager. You can create playlist audio and playlist video.

<h4>Features</h4>
<ul>
<li>Genre manager: Name, Image, Description</li>
<li>Album manager: Nam, Image, Album type, Description, Date create.</li>
<li>Artist manager: Name, Image, Description, Real name, Birthday, Occupation, Instruments, Website.</li>
<li>Audio, Video song: Name, Lyric, Description, Artist, multi album, Song for free or sale.</li>
<li>Short code to display album.</li>
<li>Compatible with Tablet, Smart phone.</li>
<li>Compatible with Visual Composer plugin.</li>
</ul>

== Installation ==

<h4>1. Automatic installation:</h4>

The simplest way to install is to click on 'Plugins' then 'Add new' and type 'Music Press' in the search field.

<h4>2. Manual installation 1:</h4>
<ol>
<li>Login to your website and go to the Plugins.</li>
<li>Click the Add New button.</li>
<li>Under Install Plugins, click the Upload Plugin.</li>
<li>Select the plugin zip file (music-press.zip) from your computer then click the Install Now button.</li>
<li>You should see a message stating that the plugin was installed successfully.</li>
<li>Click the Activate Plugin.</li>
</ol>

<h4>2. Manual installation 2:</h4>
<ol>
<li>You should have access to the server where WordPress is installed.</li>
<li>Upload the plugin zip file (music-press.zip) up to your server path(/wp-content/plugins/) and unzip it here.</li>
<li>Login to your website and go to the Plugins section of your admin panel.</li>
<li>Look for "Music Press" and click Activate.</li>
</ol>

== Frequently Asked Questions ==

= How to use short code to display album? =

Please add this short code to text editor in post or page or widget: <br>
[music_press_album album_id="Enter your album id"]
